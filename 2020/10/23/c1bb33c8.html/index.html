安装新版微信 2.9.5+

解决字体问题

从 windows 中系统盘所在的位置 C:\windows\Fonts 中拷贝全部的字体，
将拷贝的字体数据放在 ubuntu 系统中 ~/.wine/drive_c/windows/Fonts 中.

配置wineconfig

终端运行winecfg，选择Windows 版本：Windows 7
